📌 Project Overview

CMS & Cart is a structured Django-based eCommerce project that demonstrates modular design, reusable templates, and practical implementation of modern web development concepts. It is designed for learning purposes and can easily be scaled for real-world applications.


The platform enables users to browse products, manage their accounts, and perform shopping operations efficiently using Django's modular architecture.

🎯 Objectives

  • Build a modular and reusable Django project
  • Provide a complete eCommerce solution
  • Enable easy UI customization using templates
  • Support multiple apps (Accounts, Shop, Themes)
  • Ensure scalability and maintainability
